GIBM Timetable

v

This website shows timetables for the different professions and classes at the GIBM.
The data comes from a JSON interface provided by the school.

Implementing such a website was a school project for module #133.
The goal was to get some client-side web development experience by using JavaScript, jQuery, AJAX and Bootstrap.
